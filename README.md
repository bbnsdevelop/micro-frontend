# micro-frontend
Example Micro-frontend Project With Angular


* root app runs on port 4200
  cd micro-root
  npm install
  npm start
* micro header runs on port 4300
  cd micro-header
  npm install
  npm start
* micro dashboard runs on port 4202
  cd micro-dashboard
  npm install
  npm start
*  micro footer runs on port 4201
   cd micro-footer
   npm install
   npm start
